# 数字公開テンプレート

> 収益・ユーザー数などの指標を公開するフォーマット

---

## 公開する指標の選択

### 公開推奨

| 指標 | 理由 |
|------|------|
| ユーザー数 | 成長の証明 |
| MRR/ARR | 収益の透明性 |
| チャーン率 | 正直さの証明 |
| NPS | 満足度の指標 |

### 公開注意

| 指標 | 注意点 |
|------|--------|
| 利益率 | 競合に情報を与える |
| CAC | 戦略が読まれる |
| 給与 | 採用に影響 |

## レビューチェックリスト

- [ ] 数字が正確
- [ ] 比較対象（先月、前年同月など）がある
- [ ] 良い面も悪い面も含まれている
- [ ] 数字の背景説明がある
- [ ] 次のアクションが明確
- [ ] 機密情報が含まれていない
