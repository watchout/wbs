# ティーザーコンテンツテンプレート

> Pre-Pre Launchフェーズで期待感を醸成するコンテンツ

---

## ティーザーの目的

1. **興味を引く**（何かが起きる予感）
2. **リストを構築する**（事前登録を促す）
3. **話題を作る**（シェアを促進）

---

## ティーザー投稿パターン

### パターン1: 謎かけ型

```
【X投稿】
ずっと温めていたプロジェクトが、
ついに形になりそうです。

{{ヒント（抽象的）}}

詳細は来週発表。
気になる方はフォローしておいてください。
```

### パターン2: カウントダウン型

```
【X投稿 Day-7】
あと7日。

{{プロダクトのシルエット/ぼかし画像}}

{{ハッシュタグ}}

---

【X投稿 Day-3】
あと3日。

{{もう少し明確なヒント}}

事前登録で{{特典}}をプレゼント。
{{URL}}
```

### パターン3: Behind the Scenes型

```
【X投稿】
最近、こんなことに取り組んでいます。

{{作業中の写真/スクリーンショット（一部ぼかし）}}

{{何を作っているかのヒント}}

もうすぐ発表できそう。
```

### パターン4: 課題提起型

```
【X投稿】
{{ペインポイント}}で困っている人、
どれくらいいますか？

いいね/RT で教えてください。

実は、その解決策を準備中です...
```

---

## 事前登録LP構成

```markdown
## {{Coming Soon}}

### {{キャッチコピー（抽象的）}}

{{ベネフィットのヒント}}

### 事前登録で{{特典}}をプレゼント

[メールアドレス入力]
[登録する]

### {{発表予定日}}に詳細を公開します
```

---

## AI生成プロンプト

```
以下の情報を元に、ティーザーコンテンツを生成してください。

【プロダクト情報】
- 名前: {{公開しない}}
- カテゴリ: {{ヒントになる程度}}
- ベネフィット: {{抽象的に}}

【制約】
- 具体的な機能は明かさない
- 興味を引く謎を残す
- 事前登録への誘導を含める

【出力】
1. X投稿7日前
2. X投稿3日前
3. X投稿1日前
4. 事前登録LPコピー
```

---

## レビューチェックリスト

- [ ] 興味を引く謎がある
- [ ] 情報を出しすぎていない
- [ ] 事前登録への導線がある
- [ ] シェアしたくなる要素がある
- [ ] ブランドイメージと一致

| 日 | プラットフォーム | 内容 |
