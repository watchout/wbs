# ローンチメールシーケンステンプレート

> Launchフェーズのメール配信設計

---

## シーケンス概要

| 日 | メール | 目的 |
|----|--------|------|
| Day 1 | カートオープン | 販売開始告知 |
| Day 2 | 価値の再確認 | ベネフィット訴求 |
| Day 3 | FAQ | 不安解消 |
| Day 4 | 社会的証明 | 購入者の声 |
| Day 5 | 特典リマインド | 緊急性 |
| Day 6 | 24時間前 | 最終リマインド |
| Day 7 | カートクローズ | クロージング |

---

## メール1: カートオープン

```
件名: 【本日公開】{{プロダクト名}}

---

{{名前}}さん、

お待たせしました。

ついに{{プロダクト名}}の申し込みを開始します。

{{プロダクト名}}は、{{ベネフィットを1文で}}。

【{{プロダクト名}}の詳細を見る】
{{URL}}

---

特別オファーは{{期限}}までです。

{{署名}}
```

---

## メール2: 価値の再確認

```

2. {{ベネフィット2}}
3. {{ベネフィット3}}

{{URL}}

{{署名}}
```

## メール7: カートクローズ

```
件名: 【本日終了】{{時間}}に締め切ります

---

{{名前}}さん、

これが最後のメールです。

{{プロダクト名}}の申し込みは
本日{{時間}}で終了します。

{{プロダクト名}}があなたにできること:
{{ベネフィットまとめ}}

決断は{{名前}}さん次第です。

【申し込む】
{{URL}}

次の募集は{{次回の時期}}を予定しています。

{{署名}}
```

## レビューチェックリスト

- [ ] 件名が開封したくなる
- [ ] 各メールに明確な目的がある
- [ ] CTAが1つだけ
- [ ] 緊急性が段階的に高まる
- [ ] 押し売り感がない
- [ ] 配信時間が最適化されている
