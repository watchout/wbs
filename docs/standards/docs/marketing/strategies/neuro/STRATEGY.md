# ニューロマーケティング戦略

> 脳科学に基づいた感情訴求で行動を促す手法

---

## 概要と原則

### ニューロマーケティングとは

脳科学・心理学の知見を活用し、人間の無意識レベルの反応に訴えかけるマーケティング手法。
論理よりも感情が購買決定の90%を占めるという研究に基づく。

### 3つの脳と訴求

| 脳 | 役割 | 訴求方法 |
|-----|------|---------|
| 爬虫類脳 | 生存本能 | 恐怖、安全、緊急性 |
| 哺乳類脳 | 感情 | 喜び、悲しみ、共感 |
| 大脳新皮質 | 論理 | データ、証拠、比較 |

### 6つの感情トリガー

| トリガー | 説明 | 活用例 |
|---------|------|--------|
| 恐怖（FOMO） | 機会損失への恐れ | 「今だけ」「残りわずか」 |
| 所属 | 仲間意識 | 「◯◯人が利用」 |
| 権威 | 専門家への信頼 | 「◯◯監修」「受賞」 |
| 希少性 | レアなものへの欲求 | 「限定」「先着」 |
| 互恵性 | お返ししたい心理 | 無料提供→有料提案 |
| 一貫性 | 一度決めたことを守りたい | 小さなYesから始める |
