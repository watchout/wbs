# ストーリー動画台本テンプレート

> 感情に訴えかけるストーリー形式の動画構成

---

## ストーリー構成（ヒーローズジャーニー短縮版）

### 構成概要

```
1. 日常の世界（共感）
2. 問題の発生（緊張）
3. 解決策との出会い（希望）
4. 変化の達成（喜び）
5. 行動への誘い（CTA）
```

---

## 台本テンプレート

### シーン1: 日常の世界（0:00-0:30）

```
【映像】主人公の日常シーン

【ナレーション/テロップ】
「{{主人公}}は、{{普通の日常}}を送っていました。」

【感情】共感、親近感
【BGM】穏やかな日常感
```

### シーン2: 問題の発生（0:30-1:00）

```
【映像】問題に直面するシーン

【ナレーション/テロップ】
「でも、{{問題}}が起きました。」
「{{具体的な困りごと}}」

【感情】不安、焦り
【BGM】緊張感のある音楽
【演出】スローモーション、暗転
```

### シーン3: 解決策との出会い（1:00-1:30）

```
【映像】解決策を見つけるシーン

【ナレーション/テロップ】
「そんなとき、{{解決策/プロダクト}}と出会いました。」

【感情】希望、期待
【BGM】明るくなり始める
【演出】光が差し込む演出
```

### シーン4: 変化の達成（1:30-2:00）

```
【映像】変化後の様子、笑顔

【ナレーション/テロップ】
「今では、{{ポジティブな状態}}」
「{{具体的な成果}}」

【感情】喜び、達成感
【BGM】アップテンポ、明るい
【演出】ビフォーアフター比較
```

### シーン5: 行動への誘い（2:00-2:15）

```
【映像】CTA画面、URL表示

【ナレーション/テロップ】
「あなたも、{{ベネフィット}}を手に入れませんか？」
「詳しくはこちら」

【感情】行動意欲
【BGM】余韻を残す
```

## AI生成プロンプト

```
以下の情報を元に、2分のストーリー動画台本を生成してください。

【参照ドキュメント】
- users/personas.md: 主人公のモデル
- users/pain-points.md: 問題
- VALUE_PROPOSITION.md: 解決策

【構成】
ヒーローズジャーニー短縮版
1. 日常 2. 問題 3. 出会い 4. 変化 5. CTA

【制約】
- 感情の起伏を明確に
- 映像の指示を含める
- BGM/演出の指示を含める
```

---

## レビューチェックリスト

- [ ] 共感できる主人公設定
- [ ] 問題が視聴者と共通
- [ ] 感情の起伏がある
- [ ] 解決が論理的
- [ ] 視覚的に美しい
- [ ] CTAが自然
- [ ] 2分以内に収まる

## プラットフォーム別調整

### YouTube

```
- 2-5分OK
- チャプター追加
- エンドカードでチャンネル登録誘導
```

### Instagram Reels

```
- 30-60秒に凝縮
- 最初の3秒で興味を引く
- 縦型（9:16）
- テキストは大きく
```

### TikTok

```
- 15-60秒
- トレンド音源の活用
- ネイティブ感を重視
```
