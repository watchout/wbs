## プラットフォーム別指示

### Instagram投稿（1080x1080）

```
- テキスト量: 画像の20%以下
- 読みやすさ: スマホで一目で分かる
- 保存したくなる価値: Tips/データ/チェックリスト
```

### Instagramリールサムネイル（1080x1920）

```
- 最初の1秒で興味を引く
- テキストは上部1/3に配置
- 人の顔がある方がクリック率高い
```

### Facebook/Instagram広告（1200x628）

```
- テキスト20%ルールを意識
- CTAボタンとの調和
- ブランドロゴの配置
```
