# DRM（ダイレクトレスポンスマーケティング）戦略

> 即座の行動（購入、登録、問い合わせ）を促すマーケティング手法

---

## 概要と原則

### DRMとは

広告やコンテンツを通じて、ターゲットに**直接的な行動**を促すマーケティング手法。
ブランディング広告と異なり、**測定可能な成果**を重視する。

### 3つの原則

1. **明確なCTA（Call To Action）**
   - 1コンテンツ1アクション
   - 迷わせない導線設計

2. **緊急性・希少性の活用**
   - 期間限定、数量限定
   - 今行動する理由を提示

3. **リスクリバーサル**
   - 返金保証、無料トライアル
   - 行動のハードルを下げる

---

## 適用条件

### 向いているプロダクト

- 明確な価値提案があるサービス
- 価格が決まっている商品
- CVが定義しやすい（購入、登録、申込）

### 向いているフェーズ

- LP公開後のリード獲得期
- グロース期のCV最大化
- キャンペーン実施時

### プロジェクトタイプ別

| タイプ | 適合度 | 備考 |
|--------|-------|------|
| app | ◎ | メイン手法として活用 |
| lp | ◎ | LPの本質がDRM |
| hp | △ | 問い合わせ誘導に限定 |
| api | △ | 無料トライアル誘導 |
| cli | △ | ダウンロード誘導 |

## KPI定義

### 主要KPI

| 指標 | 目標値 | 測定方法 |
|------|-------|---------|
| CVR（コンバージョン率） | 2-5% | LP訪問→CV |
| CPA（獲得単価） | LTV/3以下 | 広告費/CV数 |
| ROAS（広告費用対効果） | 300%以上 | 売上/広告費 |

### チャネル別KPI

| チャネル | 主要指標 | 目標 |
|----------|---------|------|
| X | クリック率 | 1%以上 |
| Instagram | CPA | 目標CPA以下 |
| note | PV→CV率 | 1%以上 |
| メール | 開封率 | 20%以上 |
| メール | クリック率 | 3%以上 |

## スケジュールテンプレート

### 初期セットアップ（2週間）

```
Week 1:
  - Day 1-2: LP作成
  - Day 3-4: メールシーケンス作成（5通）
  - Day 5: CTA投稿テンプレート作成

Week 2:
  - Day 1-3: 広告クリエイティブ作成
  - Day 4-5: 計測設定、テスト
```

### 運用フェーズ（週次）

```
毎週:
  - CTA投稿: 3-5回/週
  - 広告最適化: 週1回見直し
  - メール配信: スケジュール通り

毎月:
  - LP改善（A/Bテスト）
  - 新規クリエイティブ追加
  - KPIレビュー
```
