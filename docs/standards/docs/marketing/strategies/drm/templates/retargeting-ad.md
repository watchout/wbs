# リターゲティング広告テンプレート

> LP訪問者への再アプローチ広告設計

---

## リターゲティング戦略

### セグメント分け

| セグメント | 条件 | 訴求内容 |
|-----------|------|---------|
| 直帰 | LP訪問、30秒以内離脱 | 価値の再訴求 |
| 検討中 | LP訪問、2分以上滞在 | 社会的証明、緊急性 |
| カート落ち | フォーム到達、未完了 | リスクリバーサル、特典 |
| 購入者 | CV完了 | アップセル、紹介 |

---

## 広告テンプレート

### 直帰セグメント向け

```
【見出し】
{{ベネフィット}}を逃していませんか？

【本文】
{{プロダクト名}}で{{成果}}を実現。
{{数字}}人以上が利用中。

【CTA】
詳しく見る

【画像】
- プロダクトのスクリーンショット
- ベネフィットを示すビジュアル
```

### 検討中セグメント向け

```
【見出し】
まだ迷っていますか？

【本文】
{{testimonialの一部}}
— {{属性}}の{{名前}}さんも愛用中

【CTA】
今すぐ始める

【画像】
- 利用者の写真（許可取得済み）
- 成果を示すグラフ
```

### カート落ちセグメント向け

```
【見出し】
あと一歩で完了です

【本文】
{{特典}}付きで{{リスクリバーサル}}。
安心してお申し込みください。

【CTA】
申し込みを完了する

【画像】
- シンプルなCTAボタン強調
- 保証バッジ
```

## AI生成プロンプト

```
以下の情報を元に、リターゲティング広告を3セグメント分生成してください。

【参照ドキュメント】
- VALUE_PROPOSITION.md: 価値提案
- users/personas.md: ターゲット

【制約】
- 見出し: 20字以内
- 本文: 50字以内
- CTA: 10字以内

【出力】
1. 直帰セグメント向け
2. 検討中セグメント向け
3. カート落ちセグメント向け
```

## レビューチェックリスト

- [ ] セグメントに応じた訴求になっている
- [ ] 見出しが目を引く
- [ ] 本文が簡潔
- [ ] CTAが明確
- [ ] 画像がメッセージと一致している
- [ ] ランディング先が適切（セグメント別）

## 配信設定

| 項目 | 設定 |
|------|------|

| フリークエンシー | 1日3回まで |
| 除外 | CV完了者（アップセル以外） |
| 配信時間 | ターゲットのアクティブ時間 |
