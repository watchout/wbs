# CTA付き投稿テンプレート

> X/SNS向けのCTA付き投稿フォーマット

---

## 基本構成

```
[フック] → [価値提示] → [CTA]
```

---

## テンプレート集

### パターン1: 問題提起型

```
{{ペインポイント}}で悩んでいませんか？

{{プロダクト名}}なら{{ベネフィット}}

{{CTA}}
{{URL}}
```

### パターン2: 数字訴求型

```
{{数字}}人が{{成果}}を達成

その秘密は{{特徴}}

詳しくはこちら
{{URL}}
```

### パターン3: 限定訴求型

```
【{{期間/数量}}限定】

{{オファー内容}}

通常{{価格}}→ 今だけ{{割引価格}}

{{URL}}
```

### パターン4: How to型

```
{{成果}}を達成する3つの方法

1. {{方法1}}
2. {{方法2}}
3. {{方法3}}

詳細は無料ガイドで解説
{{URL}}
```

### パターン5: Before/After型

```
Before: {{ビフォーの状態}}
After: {{アフターの状態}}

{{プロダクト名}}で実現

{{URL}}
```

---

## AI生成プロンプト

```
以下の情報を元に、X投稿用のCTA付きコピーを5パターン生成してください。

【参照ドキュメント】
- VALUE_PROPOSITION.md: 価値提案
- users/pain-points.md: ペインポイント
- users/personas.md: ターゲット

【制約】
- 140字以内（URLを除く）
- 1投稿1CTA
- 絵文字は最大2個

【出力】
パターン1: 問題提起型
パターン2: 数字訴求型
パターン3: 限定訴求型
パターン4: How to型
パターン5: Before/After型
```

---

## レビューチェックリスト

- [ ] フックが最初の1行にある
- [ ] 価値が明確に伝わる
- [ ] CTAが1つだけ
- [ ] URLが含まれている
- [ ] 140字以内
- [ ] ターゲットの言葉を使っている
