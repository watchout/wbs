【制約】

【トーン】
{{ターゲットに合わせたトーン}}

## レビューチェックリスト

- [ ] 件名が開封したくなる内容
- [ ] プレヘッダーが件名を補完している
- [ ] 本文が3スクロール以内で読める
- [ ] CTAが明確で1つだけ
- [ ] 配信間隔が適切
- [ ] 最終メールに緊急性がある
