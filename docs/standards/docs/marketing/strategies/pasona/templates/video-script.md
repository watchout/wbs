# 動画台本テンプレート（PASONA構成）

> YouTube向けのPASONA構成動画台本

---

## 動画構成（10分想定）

### 0:00-0:30 フック

```
【画面】タイトルカード + 衝撃的な数字/事実

【ナレーション】
「{{衝撃的な事実}}」

{{ターゲット}}の皆さん、{{問題}}で悩んでいませんか？

この動画を見れば、{{ベネフィット}}が分かります。

最後まで見てください。
```

### 0:30-2:00 Problem（問題提起）

```
【画面】問題を示すビジュアル

【ナレーション】
まず、{{問題}}について説明します。

{{問題の説明}}

こんな経験ありませんか？

- {{あるある1}}
- {{あるある2}}
- {{あるある3}}

実は、{{統計/データ}}によると、{{数字}}人がこの問題を抱えています。
```

### 2:00-3:00 Affinity（共感）

```
【画面】共感を示すビジュアル、または自分の経験

【ナレーション】
実は私も、以前は{{自分の経験}}でした。

{{具体的なエピソード}}

だから、皆さんの気持ちはよく分かります。
```

### 3:00-5:00 Solution（解決策）

```
【画面】解決策の概要図

【ナレーション】
でも、解決方法があります。

それは、{{解決策の概要}}です。

### ポイント1: {{ポイント1}}

{{説明}}

### ポイント2: {{ポイント2}}

{{説明}}

### ポイント3: {{ポイント3}}

{{説明}}
```

### 5:00-7:00 Benefit（利得）& Evidence（証拠）

```
【画面】ビフォーアフター、事例

【ナレーション】
この方法を実践すると、{{ベネフィット}}が得られます。

実際に、{{事例}}では{{成果}}を達成しました。

【testimonial or 事例詳細】
```

### 7:00-8:30 Offer & Narrowing（提案と絞り込み）

```
【画面】プロダクト紹介

【ナレーション】
今回ご紹介した解決策を、もっと簡単に実践できる方法があります。

それが{{プロダクト名}}です。

{{プロダクト名}}は、{{説明}}。

特に{{ターゲット}}の方におすすめです。
```

### 8:30-9:30 Action（行動促進）

```
【画面】CTA、URL表示

【ナレーション】
詳しくは、概要欄のリンクからご確認ください。

今なら{{特典/オファー}}もあります。

ぜひチェックしてみてください。
```

### 9:30-10:00 エンディング

```
【画面】チャンネル登録促進

【ナレーション】
この動画が参考になったら、高評価とチャンネル登録をお願いします。

次回は{{次の動画のテーマ}}についてお話しします。

それでは、また次の動画でお会いしましょう。
```

## AI生成プロンプト

```
以下の情報を元に、10分のYouTube動画台本を生成してください。

【参照ドキュメント】
- users/pain-points.md: 問題提起
- users/personas.md: ターゲット
- VALUE_PROPOSITION.md: ベネフィット
- SSOT-0_PRD.md: プロダクト情報

【構成】
PASONA形式（Problem→Affinity→Solution→Offer→Narrowing→Action）

【制約】
- 話し言葉で
- 1文は短く
- 専門用語には解説を
```

## レビューチェックリスト

- [ ] 最初の30秒で興味を引ける
- [ ] 問題が視聴者の共感を得られる
- [ ] 解決策が論理的
- [ ] CTAが明確
- [ ] 話し言葉で自然
- [ ] 視覚的な指示がある
