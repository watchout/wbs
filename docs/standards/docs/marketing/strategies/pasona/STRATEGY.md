# PASONA戦略

> 問題提起から解決への導線で行動を促すフレームワーク

---

## 概要と原則

### PASONAとは

神田昌典氏が提唱した、日本向けに最適化されたコピーライティングフレームワーク。
問題（Problem）から始まり、解決策（Solution）を提示し、行動（Action）を促す。

### 6つの要素

| 要素 | 意味 | 役割 |
|------|------|------|
| **P**roblem | 問題 | 読者の悩みを明確化 |
| **A**ffinity | 親近感 | 共感を示す |
| **S**olution | 解決策 | 解決方法を提示 |
| **O**ffer | 提案 | 具体的な商品/サービス |
| **N**arrowing | 絞り込み | 対象を限定 |
| **A**ction | 行動 | 今すぐの行動を促す |

### 新PASONA（PASBECONA）

| 要素 | 意味 |
|------|------|
| Problem | 問題提起 |

- 長文記事（note、ブログ）

## スケジュールテンプレート

### 記事連載（4週間）

```
Week 1: Problem記事（問題の深掘り）
Week 2: Affinity記事（共感と事例）
Week 3: Solution記事（解決策の提示）
Week 4: Offer記事（プロダクト紹介）
```

### 動画シリーズ（3本）

```
動画1: 問題提起（なぜこの課題が重要か）
動画2: 解決策（どう解決できるか）
動画3: 実践（具体的な始め方）
```
