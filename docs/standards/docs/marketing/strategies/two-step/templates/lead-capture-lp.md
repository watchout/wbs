# リード獲得LPテンプレート

> メールアドレス獲得に特化したLP構成

---

## LP構成（上から順）

### 1. ファーストビュー

```markdown
## {{無料コンテンツのタイトル}}

{{サブコピー: 得られる価値を1文で}}

{{無料コンテンツの画像/モックアップ}}

【無料でダウンロード】

[メールアドレス] [ダウンロード]
```

### 2. 何が得られるか

```markdown
## このガイドで学べること

✅ {{得られること1}}
✅ {{得られること2}}
✅ {{得られること3}}
✅ {{得られること4}}
✅ {{得られること5}}
```

### 3. こんな人におすすめ

```markdown
## こんな方におすすめ

- {{ターゲット特性1}}
- {{ターゲット特性2}}
- {{ターゲット特性3}}
```

### 4. 内容のプレビュー

```markdown
## 目次

1. {{章1のタイトル}}
2. {{章2のタイトル}}
3. {{章3のタイトル}}
4. ...

{{内容の一部プレビュー/スクリーンショット}}
```

### 5. 著者について

```markdown
## 著者について

{{顔写真}}

**{{名前}}**

{{簡単な経歴}}
{{なぜこのコンテンツを作ったか}}
```

### 6. CTA（繰り返し）

```markdown
## 今すぐ無料でダウンロード

[メールアドレス] [ダウンロード]

※ {{プライバシーに関する一言}}
```

---

## フォーム設計

### 最小限の項目

```
[メールアドレス] ← 必須
[ダウンロードする]
```

### 追加情報が欲しい場合

```
[メールアドレス] ← 必須
[お名前] ← 任意
[会社名/役職] ← 任意（B2B向け）
[ダウンロードする]
```

### 注意点

- 項目が増えるとCVRが下がる
- 必要最小限に
- 後からアンケートで聞く方法も

## AI生成プロンプト

```
以下の情報を元に、リード獲得LPのコピーを生成してください。

【無料コンテンツ】
タイトル: {{}}
形式: {{}}
内容: {{}}

【参照ドキュメント】
- users/personas.md: ターゲット
- users/pain-points.md: 訴求ポイント

【出力形式】
1. ファーストビューのコピー
2. 「学べること」5項目
3. 「こんな人に」3項目
4. 著者紹介文

【制約】
- シンプルで分かりやすく
- 価値を明確に
- フォームへの誘導を自然に
```

---

## レビューチェックリスト

- [ ] ファーストビューで価値が伝わる
- [ ] フォームが目立つ
- [ ] 項目数が最小限
- [ ] 無料であることが明確
- [ ] プライバシーポリシーへのリンク
- [ ] モバイルで使いやすい
- [ ] ローディングが速い

---

## A/Bテスト項目

| 要素 | テスト案 |
|------|---------|
| タイトル | ベネフィット強調 vs 好奇心喚起 |
| CTA | 「ダウンロード」vs「今すぐ読む」|
| フォーム位置 | ファーストビュー vs スクロール後 |
| フォーム項目 | メールのみ vs メール+名前 |
| 画像 | モックアップ vs 実物 |
