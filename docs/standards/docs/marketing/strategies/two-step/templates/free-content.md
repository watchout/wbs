# 無料コンテンツテンプレート

> リード獲得のための価値ある無料コンテンツ設計

---

## 無料コンテンツの種類

| 種類 | 形式 | 最適なケース |
|------|------|------------|
| Ebook/PDF | ダウンロード | 詳細な情報提供 |
| チェックリスト | 1ページPDF | 実践的なタスク |
| テンプレート | スプレッドシート等 | すぐに使える |
| 動画講座 | 動画シリーズ | 複雑な説明 |
| Webinar | ライブ/録画 | 双方向性 |
| 無料トライアル | プロダクト | 体験させる |
| ミニツール | Web/アプリ | 価値を体感 |

---

## Ebook/PDFテンプレート

### 構成（10-20ページ）

```markdown
# {{タイトル}}
## {{ターゲット}}のための{{ベネフィット}}ガイド

---

### はじめに

{{このガイドで得られること}}
{{なぜこのガイドを作ったか}}

---

### 第1章: {{問題の理解}}

{{問題の説明}}
{{なぜこれが重要か}}

---

### 第2章: {{解決策の概要}}

{{解決策のフレームワーク}}
{{ステップバイステップ}}

---

### 第3章: {{実践方法}}

{{具体的なアクション}}
{{事例}}

---

### 第4章: {{よくある間違い}}

{{避けるべきこと}}
{{対処法}}

---

### まとめ

{{重要ポイントの振り返り}}
{{次のアクション}}

---

### 著者について

{{簡単な自己紹介}}
{{プロダクトの紹介（軽く）}}

---

### 次のステップ

{{本命オファーへの誘導}}
{{CTA}}
```

## 動画講座構成

```
動画1: {{問題提起}}（5-10分）
  - 問題の重要性
  - このシリーズで学べること

動画2: {{基礎知識}}（10-15分）
  - 必要な前提知識
  - フレームワーク紹介

動画3: {{実践方法}}（15-20分）
  - ステップバイステップ
  - デモ/事例

動画4: {{次のステップ}}（5-10分）
  - まとめ
  - 本命オファーへの誘導
```

## AI生成プロンプト

```
以下の情報を元に、無料コンテンツを生成してください。

【参照ドキュメント】
- users/pain-points.md: コンテンツのテーマ
- users/personas.md: ターゲット
- VALUE_PROPOSITION.md: 核となる価値

【形式】
{{Ebook / チェックリスト / 動画講座}}

【制約】
- 本当に価値のある内容
- 出し惜しみしない
- ただし、全ては教えない（続きはプロダクトで）
- 本命オファーへの自然な導線

【出力】
{{指定された形式のコンテンツ}}
```

## レビューチェックリスト

- [ ] 無料でも十分に価値がある
- [ ] ターゲットの課題を解決している
- [ ] 読みやすい/見やすい
- [ ] 本命オファーへの導線がある
- [ ] 著者/ブランドの信頼性が伝わる
- [ ] 連絡先/URLが記載されている

## 成功パターン

### 高CVRの無料コンテンツの特徴

1. **即効性**: すぐに使える
2. **具体性**: 抽象的でない
3. **完結性**: これだけで一定の成果が出る
4. **続きがある**: もっと欲しくなる
