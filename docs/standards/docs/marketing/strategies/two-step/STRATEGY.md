# 2ステップマーケティング戦略

> 無料→有料の2段階でCVを最大化する手法

---

## 概要と原則

### 2ステップマーケティングとは

いきなり販売せず、まず**無料の価値**を提供してリードを獲得し、
信頼関係を構築してから**有料オファー**を行う手法。

### ファネル構造

```
広い入口（認知）
    ↓
無料コンテンツ（興味）
    ↓
リード獲得（メアド取得）
    ↓
ナーチャリング（信頼構築）
    ↓
有料オファー（CV）
    ↓
アップセル（LTV向上）
```

### 3つの原則

1. **価値先行**
   - 無料でも本気の価値を提供
   - 出し惜しみしない

2. **段階的な信頼構築**
   - 小さなYesを積み重ねる
   - 一貫性の法則を活用

3. **適切なタイミング**
   - 信頼が構築されてからオファー
   - 押し売りしない

## 適用条件

### 向いているプロダクト

- 高単価商品
- 説明が必要な複雑なサービス
- サブスクリプション
- 情報商材、講座

### 向いているフェーズ

- リード獲得期
- 顧客教育が必要な場合
- 高CVRを目指す場合

## スケジュールテンプレート

### セットアップ（2週間）

```
Week 1:
  - Day 1-3: 無料コンテンツ作成
  - Day 4-5: リード獲得LP作成
  - Day 6-7: メールシーケンス設定

Week 2:
  - Day 1-3: 本命オファーページ作成
  - Day 4-5: 計測設定、テスト
  - Day 6-7: 運用開始
```

### 運用フェーズ

```
継続的に:
  - 無料コンテンツの追加/更新
  - メールシーケンスの最適化
  - オファーのA/Bテスト
```
