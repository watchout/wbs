> Instagram向けのコンテンツ制約とベストプラクティス

## プラットフォーム仕様

### フィード投稿

| 種類 | サイズ | アスペクト比 |
|------|--------|-------------|
| 正方形 | 1080×1080px | 1:1 |
| 縦型 | 1080×1350px | 4:5（推奨） |
| 横型 | 1080×566px | 1.91:1 |
| カルーセル | 最大10枚 | 上記同様 |

### ストーリーズ

| 項目 | 仕様 |
|------|------|
| サイズ | 1080×1920px |
| アスペクト比 | 9:16 |
| 長さ | 最大60秒/セグメント |
| 表示時間 | 24時間 |

### リール

| 項目 | 仕様 |
|------|------|
| サイズ | 1080×1920px |
| アスペクト比 | 9:16 |
| 長さ | 15秒-90秒（3分まで可） |
| 推奨長さ | 15-30秒 |

### テキスト

| 項目 | 上限 |
|------|------|
| キャプション | 2,200文字 |
| ハッシュタグ | 30個まで |
| プロフィール | 150文字 |

## アルゴリズム特性

### 優遇される要素

| 要素 | 影響度 | 対策 |
|------|--------|------|
| 保存数 | 非常に高 | 保存したくなる価値 |
| シェア数 | 高 | DMでシェアされる内容 |
| コメント | 高 | 質問、議論を促す |
| 滞在時間 | 高 | カルーセル、長尺リール |
| リール利用 | 高 | リール重視 |

### 抑制される要素

- 外部リンク（プロフィールへ誘導）
- 低品質な画像
- 再利用コンテンツ（TikTokロゴ付き等）
- エンゲージメントベイト

## 投稿ベストプラクティス

### 投稿時間

| 曜日 | 推奨時間 |
|------|---------|
| 平日 | 7-9時、12-13時、19-21時 |
| 土日 | 10-12時、20-22時 |

### 投稿頻度

| コンテンツ | 頻度 |
|-----------|------|
| フィード | 3-5回/週 |
| ストーリー | 毎日3-7投稿 |
| リール | 3-7回/週 |

### ビジュアルのポイント

```
- 一貫したカラーパレット
- 高品質な画像（暗い/ぼやけはNG）
- 顔が映っていると反応率UP
- テキストは画像の20%以下
- 最初の画像で興味を引く
```

---

## ハッシュタグ戦略

### 使い方

```
- 投稿: 5-10個が最適
- コメントに入れる方法もあり
- 3つのカテゴリを混ぜる:
  - 大規模（100万投稿以上）: 1-2個
  - 中規模（10-100万）: 3-4個
  - 小規模（1-10万）: 2-3個
```

### ジャンル別例

```
ビジネス系:
#スタートアップ #起業 #ビジネス #仕事術

マーケティング系:
#マーケティング #SNS運用 #集客

開発系:
#プログラミング #エンジニア #開発
```

## ストーリー活用

### 活用方法

```
- 日常の共有（親近感）
- アンケート/投票（エンゲージメント）
- カウントダウン（期待感）
- リンク（誘導）
- 質問箱（双方向性）
```

### ハイライト設計

```
プロフィールに固定するカテゴリ:
- 自己紹介
- サービス紹介
- お客様の声
- よくある質問
- 最新情報
```

---

## リール活用

### リール構成

```
0-1秒: フック（スクロール止める）
1-15秒: メイン（価値提供）
15-30秒: CTA（フォロー、詳細はプロフィール）
```

### 高パフォーマンスのコツ

```
- 最初の1秒で興味を引く
- テキストオーバーレイ必須
- トレンド音源を活用
- 縦型フルスクリーン
- ネイティブ感（広告っぽくしない）
```
