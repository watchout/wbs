# X（Twitter）チャネルガイド

> X向けのコンテンツ制約とベストプラクティス

---

## プラットフォーム仕様

### 文字数制限

| アカウント | 上限 |
|-----------|------|
| 無料 | 280文字 |
| Premium | 25,000文字 |
| 日本語 | 140文字 = 280バイト |

### メディア仕様

| 種類 | 仕様 |
|------|------|
| 画像 | 最大4枚、5MB以下、JPG/PNG/GIF |
| 動画 | 最大140秒（2分20秒）、512MB以下 |
| GIF | 15MB以下 |
| サムネイル | 1200×628px推奨 |

### URL

- t.coで短縮（約23文字扱い）
- OGP画像が表示される

---

## アルゴリズム特性

### 優遇される要素

| 要素 | 影響度 | 対策 |
|------|--------|------|
| エンゲージメント率 | 高 | 質問、投票で促進 |
| 滞在時間 | 高 | 長文、スレッド |
| リプライ数 | 高 | 議論を促す |
| リポスト | 中 | シェアしたくなる内容 |
| いいね | 中 | 共感される内容 |
| ブックマーク | 高 | 保存価値のある情報 |

### 抑制される要素

- 外部リンク（リプライに入れる方が良い）
- 低エンゲージメント投稿
- スパム的な投稿パターン

---

## 投稿ベストプラクティス

### 投稿時間

| 曜日 | 推奨時間 |
|------|---------|
| 平日 | 7-8時、12-13時、18-20時 |
| 土日 | 10-12時、15-17時 |

### 投稿頻度

| 目的 | 頻度 |
|------|------|
| 認知拡大 | 3-5回/日 |
| エンゲージメント重視 | 1-2回/日 |
| 最低限 | 1回/日 |

### フック（最初の1行）

```
効果的なフック:
- 数字で始める: 「3つの方法で...」
- 疑問形: 「〜していませんか？」
- 断言: 「〜は間違いです」
- 意外性: 「実は〜」
- 共感: 「〜で悩んでいる人へ」

避けるべき:
- 長すぎる導入
- 曖昧な表現
- クリックベイト
```

## スレッド活用

### スレッドの構成

```
1/N: フック（興味を引く）
2-N-1: 本文（価値提供）
N: CTA（フォロー、RT、リンク）
```

### スレッドのメリット

- 滞在時間増加
- 保存されやすい
- 詳細な説明が可能
- シリーズ化できる

### スレッドテンプレート

```
1/5
{{衝撃的な導入}}

これを知らないと{{悪い結果}}になります。

🧵

---

2/5
まず{{ポイント1}}

{{説明}}

---

3/5
次に{{ポイント2}}

{{説明}}

---

4/5
そして{{ポイント3}}

{{説明}}

---

5/5
まとめ:

- {{ポイント1}}
- {{ポイント2}}
- {{ポイント3}}

役に立ったらRT/フォローお願いします。
```

## 分析指標

### 追跡すべきKPI

| 指標 | 計算方法 | 目標 |
|------|---------|------|
| インプレッション | - | 成長傾向 |
| エンゲージメント率 | (いいね+RT+リプ)/インプ | 3%以上 |
| プロフィールクリック | - | 投稿あたり1%以上 |
| リンククリック | - | インプの0.5%以上 |
| フォロワー増加率 | - | 週2%以上 |
