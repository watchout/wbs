## アルゴリズム特性

### 優遇される要素

| 要素 | 影響度 | 対策 |
|------|--------|------|
| 視聴維持率 | 非常に高 | 冒頭のフック、構成 |
| クリック率（CTR） | 非常に高 | サムネイル、タイトル |
| 視聴時間 | 高 | 適切な動画長 |
| エンゲージメント | 中 | いいね、コメント促進 |
| 投稿頻度 | 中 | 一貫した投稿 |

### 視聴維持率の目標

| 動画長 | 目標維持率 |
|--------|-----------|
| 5分以下 | 50%以上 |
| 10分 | 40%以上 |
| 20分以上 | 30%以上 |

## サムネイルのベストプラクティス

### 必須要素

```
1. 顔（感情が伝わる表情）
2. 大きなテキスト（3-5語）
3. コントラストの強い色
4. シンプルな構成
```

### 避けるべき

```
- 文字が小さい
- 情報量が多すぎ
- 暗い/見にくい
- 動画内容と不一致
```

### カラーパレット

```
推奨: 黄色、赤、オレンジ（目立つ）
避ける: 赤と白の組み合わせ（YouTubeのUIと被る）
```

## 動画構成テンプレート

### 基本構成（10分動画）

```
0:00-0:30  フック（問題提起、期待設定）
0:30-1:00  自己紹介、動画の価値
1:00-8:00  メインコンテンツ
8:00-9:00  まとめ、CTA
9:00-10:00 次の動画予告、エンドカード
```

### フックのパターン

```
1. 成果を見せる: 「この方法で〇〇を達成しました」
2. 問題提起: 「〇〇で悩んでいませんか？」
3. 好奇心: 「なぜ〇〇なのか知っていますか？」
4. 数字: 「3つの方法で〇〇が変わります」
5. 対立: 「〇〇は間違いです」
```

## SEO対策

### タイトル

```
- 60文字以内
- キーワードを前半に
- 数字を入れる
- 感情を喚起する言葉
```

### 説明文

```
- 最初の150文字が重要（検索結果に表示）
- キーワードを自然に含める
- チャプターを記載
- 関連リンク
- SNSリンク
```

### タグ

```
- 5-10個
- メインキーワード
- 関連キーワード
- チャンネル名
```

### チャプター

```
0:00 イントロ
1:00 {{セクション1}}
3:00 {{セクション2}}
6:00 {{セクション3}}
8:00 まとめ
```

## 分析指標

### 追跡すべきKPI

| 指標 | 目標 |
|------|------|
| CTR | 4-10% |
| 平均視聴維持率 | 40%以上 |
| 視聴時間 | 成長傾向 |
| 登録者転換率 | 視聴の1%以上 |

### アナリティクスの見方

```
1. インプレッションのCTR → サムネイル/タイトル改善
2. 視聴維持率グラフ → 離脱ポイントを特定
3. トラフィックソース → 発見経路を把握
4. 視聴者属性 → ターゲット確認
```
