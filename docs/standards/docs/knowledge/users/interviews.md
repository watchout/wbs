## インタビューガイド（テンプレート）

```
以下の質問を状況に応じてカスタマイズして使用する:

1. 背景の把握
   - 普段の業務/生活で[テーマ]についてどのように対応していますか？
   - 最後に[テーマ]で困ったのはいつですか？

2. 課題の深掘り
   - そのとき具体的にどう対処しましたか？
   - その対処法の何が不満ですか？
   - 理想的にはどうなっていてほしいですか？

3. 解決策の探索
   - [既存サービス名]は使ったことがありますか？
   - 何が良くて、何が不満でしたか？
   - もしこういう機能があったら使いますか？（仮説検証）

4. 価値の確認
   - この問題が解決されたら、どのくらいの価値がありますか？
   - お金を払うとしたらいくらくらいですか？
   - 誰かに勧めたいと思いますか？
```
