## SSOT との同期

```
この用語集の内容は以下のSSOTに反映すること:
- SSOT-0_PRD.md §用語定義
- 各機能SSOTの §2 機能概要（用語参照）
- UI文言（ユーザー向け用語を使用）

用語を追加・変更した場合:
1. この用語集を更新
2. 関連するSSOTの用語定義セクションを更新
3. UI文言に影響がある場合は該当SSOTのDETAIL層を更新
```
