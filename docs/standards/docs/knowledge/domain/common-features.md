## SSOT-1_FEATURE_CATALOG との対応

```
この知識データの機能を SSOT-1_FEATURE_CATALOG.md に反映する際:

1. 必須機能 → MVP候補として FEATURE_CATALOG に追加
2. 期待機能 → v1/v2 の候補として FEATURE_CATALOG に追加
3. 差別化機能 → VALUE_PROPOSITION と連携して優先度を判断

注意:
  知識データは「業界一般」の情報。
  自社プロダクトに必要かはSSOT-0_PRD で判断する。
```
