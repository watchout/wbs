## 1. メールマーケティング原則

### 1.1 DRM式メールの基本

```
【従来のメール】
情報提供 → 情報提供 → 情報提供 → セールス

【DRM式メール】
関係構築 → 価値提供 → 教育 → ストーリー → オファー
    ↓         ↓        ↓        ↓         ↓
  信頼      期待感    権威性    共感     行動
```

### 1.2 メールの黄金比率

| 内容 | 比率 |
|------|------|
| 価値提供（教育、ノウハウ） | 70% |
| 関係構築（ストーリー、共感） | 20% |
| オファー（販売、CTA） | 10% |

## 2. ウェルカムシーケンス（7日間）

> リード登録直後の最重要シーケンス

### 2.1 シーケンス概要

```
Day 0: 登録確認 + リードマグネット配布
    ↓
Day 1: 価値提供 + 自己紹介
    ↓
Day 2: ストーリー（共感）
    ↓
Day 3: 教育コンテンツ①
    ↓
Day 4: 教育コンテンツ②
    ↓
Day 5: 社会的証明
    ↓
Day 7: オファー（β版招待など）
```

### 2.2 各メール詳細

#### Day 0: 登録確認 + リードマグネット

| 項目 | 内容 |
|------|------|
| **目的** | 登録確認、期待値設定、即座に価値提供 |
| **送信タイミング** | 登録直後（即時） |
| **件名** | 「【ダウンロード】○○をお届けします」 |

**本文構成**:
```
[名前]さん

ご登録ありがとうございます！

お約束の「○○」をお届けします。
↓ダウンロードはこちら
[ボタン]

このあと、[何通かのメール]で
[どんな価値]をお届けします。

楽しみにしていてください！

追伸：
何か質問があれば、このメールに返信してください。
必ずお返事します。
```

## 3. ローンチシーケンス（7日間）

> 製品リリース時のシーケンス

### 3.1 シーケンス概要

```
Day -3: 予告（期待感醸成）
    ↓
Day -1: 最終予告（緊急性）
    ↓
Day 0: ローンチ告知
    ↓
Day 1: 証言・事例
    ↓
Day 3: FAQ・不安解消
    ↓
Day 5: 締め切り予告
    ↓
Day 7: 最終案内（締め切り）
```

## 4. 件名テンプレート集

### 4.1 開封率が高い件名パターン

| パターン | 例 |
|---------|-----|
| 【記号】 | 「【重要】○○のお知らせ」 |
| 質問形 | 「○○でお悩みではありませんか？」 |
| 数字 | 「○○を成功させる3つの秘訣」 |
| How to | 「○○する方法」 |
| 緊急性 | 「本日23:59まで」 |
| 好奇心 | 「これは秘密ですが...」 |
| パーソナル | 「[名前]さんへ」 |
| 証言 | 「○○さんが△△を達成した方法」 |
| 警告 | 「○○する前に読んでください」 |
| 失敗 | 「私が犯した3つの間違い」 |

### 4.2 避けるべき件名

- スパムフィルターにかかるワード（「無料」「儲かる」など）
- 全角英数字
- 長すぎる件名（40文字以上）
- 件名と本文の不一致

## 5. メール設定

### 5.1 送信設定

| 項目 | 設定 |
|------|------|
| 送信者名 | 個人名（会社名より開封率高い） |
| 送信時間 | 7:00-9:00 or 20:00-22:00 |
| 送信間隔 | 最低24時間空ける |

### 5.2 テクニカル設定

| 項目 | 設定 |
|------|------|
| SPF | 設定済み |
| DKIM | 設定済み |
| DMARC | 設定済み |
| ダブルオプトイン | 推奨 |

## 6. KPI設計

### 6.1 追跡指標

| 指標 | 目標 | 測定方法 |
|------|------|---------|
| 開封率 | 25-35% | メールツール |
| クリック率 | 3-5% | メールツール |
| 解除率 | < 1% | メールツール |
| CV率（目的達成） | 5-10% | GA + メールツール |

### 6.2 改善アクション

| 指標が低い場合 | 改善策 |
|--------------|-------|
| 開封率 < 20% | 件名をテスト、送信時間を変更 |
| クリック率 < 2% | CTAを明確に、本文を短く |
| 解除率 > 2% | 頻度を下げる、内容を見直す |
