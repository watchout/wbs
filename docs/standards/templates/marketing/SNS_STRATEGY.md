## 3. Build in Public 戦略

### 3.1 Build in Publicとは

```
従来: ステルス開発 → 完成 → 発表 → 売れない...

Build in Public:
開発の過程をオープンに共有
  ↓
共感・応援者を獲得
  ↓
リリース前からファンがいる状態
```

### 3.2 共有するコンテンツ

| カテゴリ | 内容 | 頻度 |
|---------|------|------|
| **進捗報告** | 「今週やったこと」「達成したマイルストーン」 | 週1-2回 |
| **数字の公開** | 「登録者○人達成」「売上○円」 | 月1-2回 |
| **失敗・学び** | 「やらかしたこと」「学んだこと」 | 週1回 |
| **裏側** | 「なぜこう決めたか」「ボツ案」 | 週1回 |
| **ユーザーの声** | 「こんな感想をもらった」 | 随時 |

### 3.3 Build in Public テンプレート

**週次報告テンプレート**:
```
📊 週次報告 [日付]

✅ 今週やったこと
・
・
・

📈 数字
・フォロワー: +XX (計XXX)
・β登録: +XX (計XXX)

💡 学び
・

🎯 来週やること
・

#buildinpublic #スタートアップ
```

## 4. コンテンツ戦略

### 4.1 コンテンツ4分類

```
        教育的 ←───────────→ エンタメ
           │                  │
     ┌─────┴─────┐      ┌─────┴─────┐
     │  教育系   │      │  共感系   │
     │(ノウハウ) │      │(ストーリー)│
     └───────────┘      └───────────┘
           │                  │
           ▼                  ▼
     ┌───────────┐      ┌───────────┐
     │  宣伝系   │      │ 交流系    │
     │(製品紹介) │      │(質問/会話) │
     └───────────┘      └───────────┘
           │                  │
        売り込み ←───────────→ 関係構築
```

### 4.2 コンテンツ比率

| カテゴリ | 比率 | 目的 |
|---------|------|------|
| 教育系 | 40% | 価値提供、フォロワー増 |
| 共感系 | 30% | 信頼構築、ファン化 |
| 交流系 | 20% | エンゲージメント |
| 宣伝系 | 10% | コンバージョン |

### 4.3 コンテンツ例

**教育系（40%）**:
- 「○○する5つの方法」
- 「初心者がやりがちな間違い」
- 「知らないと損する○○」
- 業界ニュースの解説

**共感系（30%）**:
- 創業ストーリー
- 失敗談
- 開発の裏側
- Build in Public

**交流系（20%）**:
- 質問への回答
- アンケート
- 他の人の投稿へのコメント
- 引用RT

**宣伝系（10%）**:
- 新機能告知
- β版募集
- ユーザーの声紹介
- LP誘導

## 7. エンゲージメント戦略

### 7.1 フォロワー獲得戦略

| 戦略 | 詳細 |
|------|------|
| **価値提供** | 有益なコンテンツを継続投稿 |
| **ハッシュタグ** | 関連タグ3-5個を使用 |
| **コミュニティ参加** | 関連コミュニティで発言 |
| **インフルエンサー** | 影響力のある人に絡む |
| **相互フォロー** | 関連アカウントをフォロー |

### 7.2 エンゲージメント向上

| 施策 | 詳細 |
|------|------|
| **質問で終わる** | 「あなたはどう思いますか？」 |
| **リプに返信** | 必ず返信する |
| **引用RT** | 他の人の投稿に付加価値 |
| **スレッド** | 長い内容はスレッドで |
| **画像/動画** | テキストより目を引く |

### 7.3 やってはいけないこと

| NG行為 | 理由 |
|--------|------|
| 宣伝ばかり | フォロー解除される |
| 自動投稿のみ | 人間味がない |
| 炎上商法 | ブランド毀損 |
| フォロワー購入 | エンゲージメント率低下 |
| 無言フォロー | 関係構築できない |
