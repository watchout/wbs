## 1. 収益最大化の原則

### 1.1 3つの成長軸（ジェイ・エイブラハム）

```
売上 = 顧客数 × 客単価 × 購買頻度

例: 各軸10%改善
100人 × ¥10,000 × 12回 = ¥12,000,000
  ↓
110人 × ¥11,000 × 13.2回 = ¥15,972,000（+33%）

例: 各軸25%改善
100人 × ¥10,000 × 12回 = ¥12,000,000
  ↓
125人 × ¥12,500 × 15回 = ¥23,437,500（+95%）
```

### 1.2 各軸の施策設計

| 軸 | 施策 | 詳細 |
|---|------|------|
| **顧客数↑** | 新規獲得 | LP、SNS、広告、紹介 |
| | CV率改善 | コピー、保証、価格テスト |
| | 解約防止 | オンボーディング、サポート |
| **客単価↑** | アップセル | 上位プラン |
| | クロスセル | 追加サービス |
| | 値上げ | 価値に見合った価格 |
| **頻度↑** | 継続率 | 習慣化、価値実感 |
| | 利用促進 | メール、通知 |

## 3. 価格決定の根拠

### 3.1 価値ベース価格設定

```
顧客が得る価値 > 支払う価格 > 提供コスト

価値の測定:
- 時間削減: ○時間/月 × 時給 = ¥○○
- コスト削減: ¥○○/月
- 売上増加: ¥○○/月
- リスク回避: ¥○○/月
                       ─────────
          総価値:      ¥○○/月

価格は総価値の10-30%が目安
```

## 4. 価格心理学

### 4.1 アンカリング

```
3つのプランを用意し、中間を選ばせる

Starter: ¥1,980/月   ← 安すぎて不安
Pro:     ¥4,980/月   ← これが選ばれやすい（推奨表示）
Enterprise: ¥19,800/月 ← アンカー（高く見せる）
```

### 4.2 端数価格

| タイプ | 例 | 印象 |
|-------|-----|------|
| 端数あり | ¥4,980 | お得感 |
| 端数なし | ¥5,000 | 高級感 |
| 極端な端数 | ¥4,999 | 安売り感（避ける） |

### 4.3 年払い割引

```
月払い: ¥4,980/月 × 12 = ¥59,760/年
年払い: ¥3,980/月 × 12 = ¥47,760/年（20%OFF）

「2ヶ月分お得！」と表現
```

## 6. 解約防止（Churn Prevention）

### 6.1 解約理由と対策

| 解約理由 | 対策 |
|---------|------|
| 使い方がわからない | オンボーディング強化 |
| 価値を感じない | 成功指標の可視化 |
| 高い | 低価格プラン提案（ダウンセル） |
| 競合に乗り換え | 乗り換えコスト提示 |
| 一時的に不要 | 休止プラン提供 |

### 6.2 解約フロー

```
解約ボタン
    ↓
解約理由アンケート
    ↓
カウンターオファー提示
（例: 「1ヶ月無料延長しますか？」）
    ↓
最終確認
    ↓
解約完了 + フィードバック依頼
```

## 8. 単位経済（Unit Economics）

### 8.1 重要指標

| 指標 | 計算式 | 目標 | 現状 |
|------|-------|------|------|
| LTV | ARPU × 平均継続月数 | | |
| CAC | 広告費 ÷ 獲得顧客数 | | |
| LTV/CAC | LTV ÷ CAC | 3.0以上 | |
| CAC回収期間 | CAC ÷ ARPU | 12ヶ月以内 | |
| 月次解約率 | 解約数 ÷ 顧客数 | 5%以下 | |

## 9. 価格テスト計画

### 9.1 A/Bテスト

| テスト | パターンA | パターンB | 期間 |
|-------|----------|----------|------|
| 価格 | ¥4,980 | ¥5,980 | 2週間 |
| 年払い割引 | 15%OFF | 20%OFF | 2週間 |
| プラン数 | 2プラン | 3プラン | 2週間 |

### 9.2 価格改定の判断基準

| 状況 | アクション |
|------|-----------|
| CV率が低い | 価格を下げる or 価値訴求を強化 |
| LTV/CACが高すぎる | 価格を上げる余地あり |
| 解約率が高い | 価格以外の問題を調査 |
