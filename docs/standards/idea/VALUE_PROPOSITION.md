## 4. フィット分析

### 4.1 Problem-Solution Fit

> 課題と解決策が合っているか

| 痛み/利得 | 対応する機能 | フィット度 | コメント |

| | | ◎ / ○ / △ / × | |
| | | ◎ / ○ / △ / × | |
| | | ◎ / ○ / △ / × | |

**凡例**:
- ◎: 完全にフィット
- ○: 概ねフィット
- △: 部分的にフィット
- ×: フィットしない
