# ミエルボード - サービス概要書

## 📋 エグゼクティブサマリー

**ミエルボード**は、現場のホワイトボードスケジュール管理をデジタル化し、AI機能とリアルタイム共有で業務効率を劇的に向上させるB2Bサービスです。

### 🎯 解決する課題
- **情報共有の非効率性**: ホワイトボードの内容が離れた場所から確認できない
- **記録の消失**: アナログ管理による履歴データの消失
- **入力の手間**: 手動でのスケジュール入力・更新作業
- **最適化の困難**: 人間の経験則だけでは最適なスケジュール調整が困難

### 💡 提供価値
- **現場ファースト**: アナログの良さを保ちながらデジタルの利便性を追加
- **AI活用**: 音声入力から最適化まで、AI技術で業務効率を向上
- **トータルサポート**: ハードウェア設置から運用サポートまで一貫提供

---

## 🌟 MVV（ビジョン・ミッション・バリュー）

### ビジョン
> **「すべての現場に、"ちょうどいい"デジタル化を。」**

### ミッション
> **「現場の"困った"を見逃さず、使いたくなるスケジュール管理を形にする。誰でも書けて、誰でも見える。アナログの良さを活かし、リアルタイム性と記録性を加える。」**

### バリュー
- **現場ファースト** - 現場の声を最優先に考える
- **使いやすさ主義** - シンプルで直感的な操作を重視
- **空気のような存在** - 自然に現場に溶け込むシステム
- **困ったの裏に宝がある** - 小さな問題から本質的な改善を見つける
- **アナログを否定しない** - 既存の良い習慣を尊重して進化

## 🚀 コア機能

### 📱 基本機能
- **週間カレンダービュー** - ホワイトボードライクなスケジュール管理
- **リアルタイム更新** - 複数デバイス間での即座な情報同期
- **履歴管理** - 1ヶ月〜3年間のスケジュール履歴保存
- **権限管理** - レベル別（1-5段階）アクセス制御
- **大型ディスプレイ対応** - タッチパネル・モニター最適化UI

### 🤖 AI機能による独自性

#### 🎤 AI音声認識入力
- **高精度音声認識**: OpenAI Whisper API活用
- **自然言語処理**: 「明日の10時から会議」→自動スケジュール登録
- **現場騒音対応**: 騒音下でも正確な音声認識を実現

#### 🧠 AIスケジュール最適化
- **過去データ学習**: 過去のスケジュールパターンを分析
- **最適化提案**: 参加者の空き時間・リソース状況を考慮した最適スケジュール提案
- **競合回避**: 時間・リソース競合の自動検出と代替案提示

#### ⏰ AIリマインド機能
- **行動パターン学習**: ユーザーの過去の行動パターンを分析
- **最適タイミング通知**: 移動時間・準備時間を考慮した適切なリマインダー
- **マルチチャネル対応**: メール・アプリ通知・SMS対応

#### 🏢 AI会議室最適化
- **利用パターン分析**: 会議室の利用実績データを分析
- **最適選択**: 参加者数・必要設備・アクセス性を考慮した会議室提案
- **効率化**: エネルギー効率・利用効率の向上

### 🔗 外部システム連携
- **Googleカレンダー**: 双方向同期対応
- **Office365**: Microsoft環境との統合
- **CRM連携**: 顧客管理システムとの連携（プレミアムプラン）

## 🏗 技術アーキテクチャ

### 💻 システム構成
- **フロントエンド**: Nuxt 3 + Vue 3 + Pinia
- **バックエンド**: Node.js + Nuxt 3 Server API
- **データベース**: PostgreSQL + Prisma ORM
- **AI連携**: OpenAI Whisper API + GPT-4
- **リアルタイム通信**: WebSockets
- **インフラ**: ConoHa VPS

### 🔒 セキュリティ・信頼性
- **データ暗号化**: 通信・保存データの暗号化
- **バックアップ**: 自動バックアップ（日次・週次）
- **アクセス制御**: レベル別権限管理
- **監査ログ**: 全操作の記録・追跡可能
- **可用性**: 99.9%以上のサーバー稼働率

### 📱 デバイス対応
- **PC**: Windows・Mac対応
- **タブレット**: iPad・Android対応
- **スマートフォン**: iOS・Android対応
- **大型ディスプレイ**: タッチパネル・モニター最適化
