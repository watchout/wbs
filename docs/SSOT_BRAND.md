```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│  「現場の"見えない"を、"見える"に。」                          │
│                                                              │
│  現場で起きるトラブルの多くは、「見えない」ことが原因。         │

│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 5-2. エレベーターピッチ

**30秒版**:
> ミエルシリーズは、現場の「見えない」を「見える」に変えるツールです。
> 誰がどこにいるか、在庫がいくつあるか、車両の管理状況…
> これまで電話やLINEで確認していた情報が、スマホやTVで一目でわかるようになります。
> 導入初日から使えて、今の運用を壊しません。

**10秒版**:
> 現場の予定・在庫・車両が一目でわかるツールです。
> 電話確認が減り、ミスが減ります。

### 5-3. ターゲット別メッセージ

| ターゲット | メッセージ | 訴求ポイント |
|-----------|-----------|-------------|
| **経営者** | 「現場の動きが、経営の見える化に。」 | 可視化・意思決定支援 |
| **配車担当者** | 「朝の電話確認、もう必要ありません。」 | 業務効率化 |
| **現場作業員** | 「事務所に戻らなくても、予定がわかる。」 | 利便性 |
| **IT担当者** | 「設定不要、研修不要。すぐ使えます。」 | 導入の簡単さ |

## 6. トーン＆マナー

### 6-1. ブランドパーソナリティ

| 特性 | 説明 | 表現例 |
|------|------|--------|
| **親しみやすい** | 難しい言葉を使わない | 「見える」「わかる」「すぐに」 |
| **頼れる** | 信頼感・安心感 | 「任せてください」「確実に」 |
| **シンプル** | 過剰な装飾をしない | 余白を活かしたデザイン |
| **現場目線** | 現場の言葉を使う | 「現場」「職人」「配車」 |

### 6-2. NG表現

| NG | 理由 | OK |
|----|------|-----|
| 「革命的」「画期的」 | 大げさ・信頼性低下 | 「便利な」「役立つ」 |
| 「AI搭載」の強調 | 機能より価値を伝える | 「話すだけで入力完了」 |
| 「業界初」「No.1」 | 根拠なし | 具体的な数値・事例 |
| 「DX」「デジタル変革」 | 抽象的・現場に響かない | 「見える化」「電話が減る」 |

| 項目 | ルール |
