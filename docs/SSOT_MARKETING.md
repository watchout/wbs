## 3. USP（Unique Selling Proposition）

### 3-1. メインUSP ✅ 確定

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│  「現場の"見えない"を、"見える"に。」                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**コンセプト**:
現場トラブルの大半は「見えない」ことが原因。
- 人の予定が見えない
- 在庫の有無・過不足が見えない
- 車両関連の法務遵守状況が見えない
- 図面の変更点、関係各所への認識が見えない

これらが「見える」ようになることで、トラブルを未然に防ぐ。

### 3-2. サブUSP（製品別）✅ 確定

| 製品 | サブUSP | 見えるようになるもの |
|------|---------|-------------------|
| **ミエルボード** | 予定がミエル | 誰がどこで何をしているか |
| **ミエルストック** | 在庫がミエル | 資材の有無・過不足 |
| **ミエルドライブ** | 車両がミエル | 配車・アルコールチェック状況 |
| **ミエルファイル** | 変更と履歴がミエル | 図面・資料の変更検知・施工履歴 |

### 3-3. タグライン

```
「見えないから起こるトラブル、ミエルで終わりにする。」
```

### 3-4. ブランドプロミス（3つの約束）

| 約束 | 内容 |
|------|------|
| **1. 見える** | 必要な情報がいつでもどこでも見える |
| **2. 簡単** | 導入初日から全員が使える |
| **3. 安心** | 今の運用を壊さず、進化させる |

## 6. パートナー・紹介制度 ✅ 確定

### 6-1. 基本方針

**ターゲットから逆算したパートナー戦略**

ミエルシリーズのターゲット（弱電・電気工事会社 10〜50名）に日常的に接点があるプレイヤーをパートナーとして開拓する。

### 6-2. パートナー体系

```
┌─────────────────────────────────────────────────────────────┐
│  ミエルシリーズ パートナー制度                                 │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  👥 ユーザー紹介（既存顧客向け）                 【Phase 0〜】 │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━              │
│  対象: ミエルシリーズ利用企業                                  │
│  動機: 「うちも使ってるよ」と業界仲間に紹介                    │
│  報酬: 初回契約額の20%（最大¥50,000）                         │
│  特典: 紹介成立で自社の翌月利用料10%OFF                        │
│  条件: 被紹介者が3ヶ月以上継続利用                             │
│                                                              │
│  🏭 電材卸・商社パートナー                       【Phase 2〜】 │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━              │
│  対象: 電材卸、工具商社、電設資材販売店                        │
│  動機: 取引先への付加価値サービス                              │
│  報酬: 月額の15%×12ヶ月                                       │
│  特典: 共同販促、営業ツール提供                                │
│  例: 渡辺パイプ、因幡電機産業、地域電材卸                      │
│                                                              │
│  🏛️ 業界団体パートナー                          【Phase 3〜】 │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━              │
│  対象: 電気工事業組合、弱電協会                                │
│  動機: 会員向けサービス向上                                    │
│  報酬: 会員向け特別価格（15%OFF）+ 紹介料10%                  │
│  特典: 団体名義での導入事例掲載                                │
│                                                              │
│  📋 士業パートナー                               【Phase 3〜】 │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━              │
│  対象: 建設業専門の税理士、社労士、行政書士                    │
│  動機: 顧問先への業務改善提案                                  │
│  報酬: 月額の10%×12ヶ月                                       │
│  特典: 顧問先への特別価格提供権                                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 6-3. Phase別ロードマップ

| Phase | 期間 | 施策 | 目標 |
