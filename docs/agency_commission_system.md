# 代理店手数料体系仕様書

## 📊 **手数料体系概要**

### **基本原則**
- **明確性**: 計算方法が明確で予測可能
- **公平性**: 努力に応じた適正な報酬
- **シンプル性**: 複雑な変動要因を排除

---

## 💰 **手数料体系詳細**

### **1. 基本手数料率（固定）**

| 代理店レベル | 基本手数料率 | 月間売上目標 | 初期費用 |
|-------------|-------------|-------------|----------|
| **ブロンズ** | **25%** | なし | 5万円 |
| **シルバー** | **30%** | 50万円/月 | 10万円 |
| **ゴールド** | **35%** | 100万円/月 | 20万円 |
| **プラチナ** | **40%** | 200万円/月 | 50万円 |

### **2. 継続報酬期間（固定）**

- **全レベル共通**: **24ヶ月間**
- **新規顧客**: 初回契約から24ヶ月間の月額料金の手数料率分
- **プラン変更**: 変更後のプランの差額分に対して手数料適用

---

## 🎯 **手数料計算例（実際のプラン）**

### **ケース1: ブロンズ代理店 - ライトプラン顧客**
```
顧客: ライトプラン (月額4,980円)
手数料率: 25%
月額手数料: 4,980円 × 25% = 1,245円
24ヶ月総額: 1,245円 × 24ヶ月 = 29,880円
```

### **ケース2: ゴールド代理店 - プレミアムプラン顧客**
```
顧客: プレミアムプラン (月額19,800円)
手数料率: 35%
月額手数料: 19,800円 × 35% = 6,930円
24ヶ月総額: 6,930円 × 24ヶ月 = 166,320円
```

### **ケース3: プラチナ代理店 - エクストラプラン顧客**
```
顧客: エクストラプラン (月額39,800円)
手数料率: 40%
月額手数料: 39,800円 × 40% = 15,920円
24ヶ月総額: 15,920円 × 24ヶ月 = 382,080円
```

## 🚀 **ボーナス制度（追加報酬）**

### **月間成果ボーナス**
- **5件以上成約**: +10,000円/件
- **10件以上成約**: +15,000円/件
- **20件以上成約**: +25,000円/件

### **年間実績ボーナス（現実的な目標）**
- **年間売上200万円達成**: 20万円
- **年間売上500万円達成**: 50万円  
- **年間売上1,000万円達成**: 100万円

---

## 📅 **支払いスケジュール**

### **支払いタイミング**
- **顧客の初回支払い確認後**: 5営業日以内
- **毎月の継続支払い**: 顧客支払い確認後3営業日以内

### **支払い方法**
1. 銀行振込（手数料当社負担）
2. PayPal（海外代理店対応）

## 🔄 **レベルアップ条件（実際のプランベース）**

### **ブロンズ → シルバー**
- 月間8件以上の成約を3ヶ月連続達成
- または月間売上10万円を3ヶ月連続達成
- 追加初期費用: 5万円

### **シルバー → ゴールド**
- 月間15件以上の成約を3ヶ月連続達成
- または月間売上20万円を3ヶ月連続達成
- 追加初期費用: 10万円

### **ゴールド → プラチナ**
- 月間25件以上の成約を3ヶ月連続達成
- または月間売上35万円を3ヶ月連続達成
- 追加初期費用: 30万円

## 🛡️ **契約条件・注意事項**

### **契約期間**
- 最低契約期間: 12ヶ月
- 自動更新: あり（1年間）
- 解約通知: 3ヶ月前

### **禁止事項**
- 虚偽の顧客情報提供
- 競合他社との同時契約
- 価格の無断変更・割引提供

### **サポート体制**
- 専任担当者によるサポート
- 月1回の売上レビューミーティング
- マーケティング素材・ツールの提供
